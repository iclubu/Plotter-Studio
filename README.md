# 🎨 Universal Plotter Studio

An interactive Python studio for generating pen-plotter ready generative art.

## Features
* **FMM Topography**: Perfect for high-contrast characters like Darth Vader.
* **Contour Streamlines**: Advanced flow-field logic for hand-drawn pencil sketches.
* **Structured Squiggles**: Artistic line shading with variable frequency.

## Setup
1. Clone the repo
2. Activate venv: `source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `streamlit run fmm_studio.py`
